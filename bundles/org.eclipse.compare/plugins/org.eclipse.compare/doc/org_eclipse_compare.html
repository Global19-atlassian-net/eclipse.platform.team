IBM Eclipse Platform Compare

This document lists all of the extension points that the compare plug-in makes available to tool developers.

All XML sub-elements defined in the individual extension point documents may appear more than once inside an extension element. For example, an org.eclipse.compare.contentMergeViewers extension may contain several instances of viewer elements. Although making one extension per sub-element is not technically incorrect, we recommend grouping for consistent style. It can also improve platform startup time because there will be fewer extensions to process.

Extension Points

The following extension points can be used to extend the capabilities of the compare infrastructure:
  • org.eclipse.compare.contentMergeViewers
  • org.eclipse.compare.contentViewers
  • org.eclipse.compare.structureMergeViewers
  • org.eclipse.compare.structureCreators
