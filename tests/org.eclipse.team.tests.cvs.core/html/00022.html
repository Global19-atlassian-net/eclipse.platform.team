Persistance and deletion

Since:
Last Modified: $Date: 2003/12/04 21:43:57 $

  • Select a project and perform a merge.
  • Ensure that the merge appears in the Synchronize View
  • Exit and restart eclipse
  • The merge should still exist in the Synchronize View
  • Delete the merge from the synchronize view using the remove toolbar button
  • The merge subscriber should be removed from the view.
